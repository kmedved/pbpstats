# Historic Backfill Runs

This directory is for generated backfill outputs, scratch reruns, validation
bundles, and local smoke-test artifacts. Contents are intentionally gitignored.

Use timestamped or descriptive subdirectories, for example:

```text
historic_backfill/runs/full_history_1997_2020_YYYYMMDD_label/
historic_backfill/runs/smoke_0020400335_YYYYMMDD/
```

Commit small release contracts under `historic_backfill/releases/` instead of
committing generated run directories.
